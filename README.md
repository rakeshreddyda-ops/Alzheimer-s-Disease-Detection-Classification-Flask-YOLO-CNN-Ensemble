# Alzheimer-s-Disease-Detection-Classification-Flask-YOLO-CNN-Ensemble
Flask web app for Alzheimer's detection using YOLO (object detection) and an Ensemble CNN model (99.1% accuracy). Compared 17 models including InceptionV3, Xception, ResNet, DenseNet, VGG19, NASNetMobile, and custom CNNs with Adam, SGD, and Nadam optimizers on MRI brain scans across 4 dementia classes
